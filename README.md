# FlowFit Client

Mobile-first CRM application for fitness coaches, built with Ionic + Angular.

## Tech Stack

- **Ionic 8** — Cross-platform UI framework
- **Angular 20** — Frontend framework
- **Angular Signals** — State management
- **Outfit** — Google Font (typography)

## Quick Start

### Prerequisites

- Node.js 25+
- npm
- Ionic CLI (`npm install -g @ionic/cli`)

### Setup

```bash
# 1. Clone the repository
git clone https://github.com/<your-username>/flow-fit-client.git
cd flow-fit-client

# 2. Install dependencies
npm install

# 3. Start development server
npm start
```

### Available Scripts

| Command          | Description                              |
| ---------------- | ---------------------------------------- |
| `npm start`      | Start Ionic dev server (port 8100)       |
| `npm run build`  | Production build                         |
| `npm run lint`   | Run ESLint                               |
| `npm run test`   | Run unit tests                           |
